# Todo List

简洁美观的 Todo List Web 应用，帮助你管理日常任务。

## 功能特性

- 添加新任务
- 标记任务完成/待完成
- 删除任务
- 清理已完成任务
- 实时统计待完成和已完成任务数量
- 数据持久化存储（localStorage）
- 任务文本过长时自动截断并显示提示
- 响应式设计，支持移动端

## 技术栈

- HTML5
- Tailwind CSS
- localStorage

## 如何使用

直接在浏览器中打开 `index.html` 文件即可使用。
